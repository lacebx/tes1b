# North Travels — Multi-Page Travel Website

This is a full multi-page HTML/CSS website built for a fictional travel agency called **North Travels** — a company that runs religious and historical tours to the Holy Lands and international museum destinations.

Built as part of my early web development practice, this project was my first real attempt at putting together a multi-page site with a shared stylesheet, a navigation header across all pages, an image gallery, and a tour request form. Every page connects. Every link works. The CSS is hand-rolled.

> *Birarenze* — making something that looks like a real website for the first time slaps.

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with featured tours and mission quote |
| Tours | `tours.html` | Detailed tour listings with dates and destinations |
| Request | `request.html` | Tour booking request form |
| About | `about.html` | About the agency |

---

## Tours Featured

- **See Israel** — April 15-23 (9 days)
- **See Holy Lands** — June 1-22 (21 days, covering Israel, Jordan, Egypt, Greece & Turkey)
- **See Museums** — September 1-14 (14 days, London, Paris, Athens, Cairo & Jerusalem)

---

## Tech Stack

- **Markup:** HTML5
- **Styling:** CSS3 (inline + external `header.css`)
- **No frameworks** — pure HTML/CSS, built from scratch
